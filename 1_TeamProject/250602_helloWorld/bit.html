Morse Code & Bit

점과 선, 0과 1로 시작된 디지털 언어

모스부호[Morse Code]란?

  • 짧은 신호(점, .)와 긴 신호(선, -)를 조합하여 문자를 표현하는 신호 방식입니다.
  • 주로 전신 통신에 사용되었으며, 현재는 비상 신호(SOS) 등에 사용됩니다.
  • 점과 선의 조합으로 알파벳, 숫자, 특수 문자를 표현합니다.

비트[Bit]란?

  • 컴퓨터에서 데이터를 표현하는 가장 기본적인 단위, 데이터를 처리하는 가장 작은 단위입니다.
  • 모든 컴퓨터 데이터(텍스트, 이미지, 오디오 등)는 비트로 표현됩니다.
  • 0과 1 두가지 값으로 이루어져 있습니다.

모스부호와 비트의 차이점

신호방식

모스 부호는 점과 선으로 신호를 표현하는 반면, 비트는 0과 1로 데이터를 표현합니다.

사용범위

모스 부호는 주로 전신 통신에 사용된 반면, 비트는 컴퓨터 시스템에 모든 데이터를 처리하는데 사용됩니다.

복잡도

비트는 0과 1 두가지 값으로 이루어져 있어 매우 단순한 반면, 모스 부호는 점과 선의 다양한 조합으로 문자를 표현하기 때문에 복잡합니다.
